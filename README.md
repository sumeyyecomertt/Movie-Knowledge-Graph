# Movie-Knowledge-Graph

This project is developed as part of the Knowledge Engineering and Ontologies course. It aims to design and implement a knowledge engineering solution for modeling the cinema domain using ontology engineering and Semantic Web technologies.

# Project Overview

The goal is to create an OWL-based ontology that represents relationships between movies, actors, directors, and genres. This knowledge graph will serve as a foundation for semantic search and integration with Large Language Models (LLMs) to support reliable question-answering systems.

# Objectives for the Initial Phase

In this first iteration, we have completed the following tasks:
Domain Selection: Cinema and Movie Relationships.
ORSD Preparation: Defined the purpose, scope, and target users.
Competency Questions: Established a set of 5-8 questions that the ontology should answer.
Initial Ontology Development: Created core classes (Movie, Person, Actor, Director, Genre) and relationships (directedBy, actedIn, hasGenre) in Protégé.

# Initial Ontology Structure

TBox (Conceptual Schema): Includes hierarchical class definitions and property constraints.

ABox (Instance Data): Contains initial examples such as Inception and Interstellar to demonstrate the modeling approach.

# Tools Used

Protégé: For OWL ontology modeling.
GitHub: For project documentation and version control.
